Haan, **ye feature bahut strong hai**. Professional trading journals mein goal tracking hoti hai, lekin tum ise aur practical bana sakte ho.

Main is tarah design karunga:

---

# Monthly Goal

User pehli baar target set kare.

Example:

* Monthly Return Target: **3%**
* Ya optional (skip)

---

# Automatic Calculation

Agar target **3% per month** hai, app automatically calculate kare:

### Daily Target (Compound)

Formula:

[
\text{Daily}=(1+0.03)^{1/30}-1
]

Result:

**≈ 0.0986% per day**

---

### Weekly Target

Formula:

[
(1+0.03)^{7/30}-1
]

Result:

**≈ 0.69% per week**

**Ek aur suggestion:** sirf **calendar days (30)** use mat karo. Professional trading ke liye **Trading Days** ka option do (default **22 trading days/month**). Agar user weekend par trade nahi karta, to daily target uske liye zyada realistic hoga. Isliye Settings mein option rakho:

* **Calendar Days (30)**
* **Trading Days (22)**

Isse GROEAX aur bhi professional lagega.
