Is UI ko dekh kar ek cheez clear hai: **calculation logic abhi professional trading journal jaisa nahi hai.**

Example:

* Starting Balance = **$10**
* Current Balance = **$3,324**
* Net Profit = **$3,314**

Ye values mathematically sahi hain.

Lekin kuch metrics galat calculate ho rahe hain.

### 1. Profit Factor

Agar:

* Gross Profit = $3314
* Gross Loss = $0

To Profit Factor technically **∞ (Infinity)** hoga.

Display:

> **∞**
>
> Perfect

Na ke random number.

### 9. Dashboard Formula

Har metric backend se calculate ho.

```
Net PnL
Current Balance - Starting Balance

Return %
(Net Profit / Starting Balance) ×100

Win Rate
Wins / Total Trades ×100

Profit Factor
Gross Profit / Gross Loss

Average Win
Total Winning Profit / Winning Trades

Average Loss
Total Losing Loss / Losing Trades

Average RR
Sum(RR)/Trades

Average Return
Total Return% / Trades

Expectancy
(Win Rate × Avg Win)
-
(Loss Rate × Avg Loss)

Consistency Score
Risk + RR + Position Size + Discipline

Max Drawdown
Largest Equity Drop

Recovery Factor
Net Profit / Max Drawdown

Sharpe Ratio

Sortino Ratio

Calmar Ratio
```

## Recommendation

Agar tum **Groeax ko TradeZella level** banana chahte ho, to sirf 4 metrics nahi rakhne chahiye.

Professional dashboard mein ye hone chahiye:

* Net P&L
* Return %
* Win Rate
* Profit Factor
* Average R:R
* Expectancy
* Average Win
* Average Loss
* Average Return/Trade
* Max Drawdown
* Recovery Factor
* Consecutive Wins
* Consecutive Losses
* Consistency Score
* Monthly Goal Progress
* Daily Target
* Weekly Target
* Equity Curve
* Balance Curve
* Drawdown Curve
* Calendar Heatmap
* Performance Insights (AI)

Ye structure Groeax ko beginner journal se nikal kar **professional institutional-level trading journal** bana dega.
